### d3bq

Welcome to d3bq, the best companion app to any BBQ tour in the Southeastern U.S.

## Requirements

- [Node.js](http://nodejs.org/)
- [Bower](http://bower.io/)

## Running locally

    $ git clone git@github.com:petergcutler/d3bq.git
    $ cd d3bq
    $ npm install
    $ bower install
    $ nodemon
